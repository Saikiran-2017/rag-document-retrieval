# Knowledge Assistant - Web UI

Next.js chat frontend for the FastAPI backend. See the **repository root [README.md](../README.md)** for full setup, environment variables, Docker, and deployment.

```bash
cp .env.example .env.local
npm install
npm run dev
```

Default API: `http://127.0.0.1:8000` (override with `NEXT_PUBLIC_API_URL`).
